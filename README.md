# Overwatch overlay
## Features

- Show current and start SR
- Autoupdate SR (1 rpm)
- Can show 1 or all 3 roles SR
## Query Params

| Param | Example |
| ------ | ------ |
| user | User-123 (BattleNet User#123) |
| typeRole | dd, sup, tank or empty |

## Examples
```sh
https://ow-profile-tracker.web.app/?user=Sturmvogel-21637
```
```sh
https://ow-profile-tracker.web.app/?user=Sturmvogel-21637&typeRole=sup
```
